# Mini‑Chalice (Browser Prototype)

Lightweight scaffold for our Massive‑Chalice‑inspired tactics game.
Built with **Vite** + **Phaser 4** (beta).

## Quick Start
```bash
# install deps
npm install

# run dev server
npm run dev

# build for production
npm run build
```
*Requires Node >= 18.*
